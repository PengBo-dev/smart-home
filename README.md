# 智家 AINO · 智能家居控制中心

> 人机交互课程综合作业 · 彭博 · 2309060309

一个纯前端、单文件实现的智能家居控制中心界面，内嵌多项人机交互（HCI）设计原理（隐喻、反馈、多模态、认知负荷、Nielsen 启发式、Fitts 定律、直接操纵等，代码中以 `[HCI:xxx]` 标注，便于对照报告）。

整个应用就一个 `index.html`：所有 HTML / CSS / JS 全部内联，**不依赖任何外部 CDN 或网络资源**，可离线运行。

---

## 🚀 运行方式

提供两种方式，任选其一。

### 方式一：在线访问（推荐，无需安装）

直接在浏览器打开：

**👉 https://pengbo-dev.github.io/smart-home/**

- GitHub Pages 为 **https 安全环境**，**语音控制功能开箱即用**。
- 推荐使用 **Chrome / Edge** 最新版浏览器。

> 若上面链接打不开，说明仓库尚未开启 Pages，开启方法：
> 仓库页面 → **Settings** → 左侧 **Pages** → Source 选 `Deploy from a branch` → Branch 选 `main` / `(root)` → Save，等 1～2 分钟即可访问。

### 方式二：本地运行

适合离线演示或查看源码。

1. 下载本仓库（`Code → Download ZIP`）或 `git clone`，解压。
2. 在文件夹内启动本地服务器（需已安装 [Node.js](https://nodejs.org/)）：

   ```bash
   node _serve.cjs
   ```

3. 浏览器打开 **http://localhost:8080/** 。
4. 停止服务：终端按 `Ctrl + C`。

> **关于直接双击 `index.html`：** 界面与绝大多数交互可正常使用，但 **语音控制不可用**——浏览器要求麦克风必须在「安全上下文」（https 或 localhost）下才授权。所以体验语音功能请用 **方式一** 或 **方式二**，不要用 `file://` 直接打开。

## 🧩 主要功能

- 灯光 / 空调 / 窗帘等设备的开关与连续调节（旋钮、滑块、直接操纵）
- 情景模式一键编排（多设备宏状态）
- 即时多模态反馈：视觉动效 + WebAudio 音效 + Toast 文字 + 环境光氛围
- 中文语音控制（支持中文数字，如「百分之五十」「二十四度」、空调制冷/制热/自动模式切换等）

## 🛠 技术栈

原生 HTML + CSS + JavaScript（无框架、无构建步骤）；语音基于浏览器 Web Speech API；音效基于 Web Audio API。
